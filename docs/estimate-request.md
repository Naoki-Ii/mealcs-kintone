# UIデザイン見積り依頼

## 依頼内容

給食製造業向け「ラベル印刷システム」のUI/UXデザイン（Figma）

## 概要

- **業種**: 給食製造業（病院・介護施設向け）
- **ユーザー**: 栄養士・製造担当者（非エンジニア、毎日使用）
- **現行システム**: Windows Forms風のデスクトップアプリ
- **リニューアル後**: モダンなWebアプリケーション

## 画面数

| 画面       | 内容                                                   |
| ---------- | ------------------------------------------------------ |
| 一覧画面   | 製造日ごとの印刷対象を表示・選択（2ペイン構成）        |
| 献立表画面 | 施設ごとの献立・禁食・数量を設定（2カラム + モーダル） |

| モーダル       | 内容                       |
| -------------- | -------------------------- |
| 印刷プレビュー | 印刷前の確認・微調整       |
| 禁食数量入力   | 禁食対応時の数量入力・確定 |
| 追加献立登録   | 特別メニューの登録         |

**計: 2画面 + モーダル3個**

## 希望するデザインの方向性

- シンプル・見やすい（情報量が多いため）
- 業務システム向け（派手すぎない）
- kintone風のモダンなデザイン

## 相談したいこと

- 2カラム構成でのタブレット対応
- モーダルの使い方・アニメーション
- 複数選択時の操作性（チェックボックス）
- データが多い一覧画面の情報整理
- フィードバック（成功/警告/エラー）の見せ方
